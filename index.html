Taller de HTML y JavaScript

Objetivo

Para este taller debe integrar los temas vistos en clase: HTML y JavaScript.

Planteamiento

Diseñe al menos tres paginas web, permitiendo que se pueda navegar entre ellas a través de enlaces. Cada pagina debe manejar color de fondo, imágenes, tablas, viñetas, audios, videos, texto, etc.

En una de estas paginas debe haber un formulario de registro de todos los datos de un estudiante. Este formulario debe validarse con JavaScript.

Adicionalmente, integre 10 efectos diferentes de JavaScript, en áreas de: botones, ventanas, colores, texto, mouse, etc.

Observaciones

  • El diseño y distribución debe ser usable, agradable a la vista del usuario, con un tema interesante que se desarrolle en las tres páginas.
  • Se debe desarrollar en grupo, y entregarse en el link del taller, en la fecha estipulada.
  • Si no se registran los nombres de los integrantes, la nota será asignada a la persona que subió el envío del taller.
  • Deben usarse imágenes, audios y videos de acuerdo al tema desarrollado.
  • Los efectos de JavaScript deben ser claramente diferenciables.
  • Debe subirse el proyecto completo de Java/Html.
